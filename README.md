# $K$-Means Clustering Algorithm in Graph Machine Learning

This repository is about the Research Data Foundations Camp final project which is due by April 1st, 2026. The final project consists of three elements: hypothesis, Python code output, and Data set evaluation. The topic of this project is related to graph machine learning, specifically known as the $K$-means clustering algorithm. The $K$-means clustering algorithm is an unsupervised machine learning algorithm that partitions a dataset into $K$ distinct, non-overlapping clusters.
